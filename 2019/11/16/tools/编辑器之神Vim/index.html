编辑器之神Vim

在linxu下，不得不提的一款效率工具就是Vim。学习使用vim这个工具就像学自行车，学会了想咋骑就咋骑，想多优雅就有多优雅。

1. 普通模式

首次打开 vim 就是普通模式，是使用频率最高的，所以是默认模式。

内容浏览，移动光标，翻页

j k h l 向下、上、左、右移动光标

w 单词的开头移动
e 单词的结尾移动

gg、1G 移动光标到第一行
G 、$ 、[End] 移动光标到最后一行

< ctrl + f > [PgUp] 向下一页
< ctrl + b > [PgDn] 向上一页

选择操作对象-可视模式

v , < shift + v > , < ctrl+ v > 分别对应 “字、行、列”，然后再移动以扩大选区。

– VISUAL – 字符

– VISUAL LINE – 行

– VISUAL BLOCK – 列

Esc 退出选择

剪切、复制、粘贴、删除

dd 剪切光标所在的行 10dd 剪切10行
yy 复制光标所在的行

p 在光标所在的下一行粘贴,
P 在光标所在的行粘贴

dG 删除光标所在到最后一行内容
c 重复删除多个数据
x 向后删除
X 向前删除

dw 删除单词

撤销和重做

u 撤销前一个操作
< ctrl + r > 重做前一个操作

查找

/word 向后查找word
?word 向前查找word

n 向下继续查找 N 向上继续查找

2. 插入模式-INSERT

i 从目前光标处输入内容，I 从目前所在行输入
o 从目前光标所在的下一行输入内容， O从目前光标所在的上一行输入
a 从目前光标所在的下一个字符输入内容，A从光标所在行的最后一个字符输入
r 单次编辑光标处的单个字符，一直取代光标所在的字符直到esc
Esc 退出插入编辑模式

3. 命令行模式

命令输入完成后按确定键（Enter）

单文件操作

:w 保存
:w! 强制保存
:q 离开vim
:q! ZQ 强制离开不保存
:wq ZZ 保存离开vim

:w [文件名] 另存为新文件
:e [文件名] 切换到另一个文件

多文件操作

同一个窗口 buffer 切换

使用 :ls 列举当前缓冲区，使用:b n（:b buffer_name）跳转到缓冲区

打开的文件切换，不需要推出vim。 :e文件名

多窗口切换 Windows

< ctrl + w > s 水平分割，< ctrl + w > v 垂直分割。

:vs 水平分割，:sp 垂直分割。

< ctrl + w > w h j k l 切换分屏

4. Vim偏好设置

使用指令临时设置

:set nu 显示行号
:set nonu 取消显示行号
:syntax on 语法高亮

:colorscheme 显示当前主题配色
:colorscheme < ctrl + d > 显示所有配色
:colorscheme z 配色名 修改配色名称,可以table补全
